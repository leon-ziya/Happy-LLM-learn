这是什么？

PROJECT · 一本会动的 LLM 科普书

📖 让大语言模型「看得见、摸得着、学得会」

传统教材是「静态」的：密密麻麻的文字、平铺的流程图、难以感知的维度变化。Happy-LLM-Learn 是一次「视觉化学习」实验——我们借助 AI 的结构化能力，将《Happy-LLM》中晦涩的概念拆解为逐帧动画、时间轴、组件拆解图，封装进一个个独立的 HTML 文件中。

简单说：这是一本「会动」的 LLM 科普书。 点一下「下一步」，复杂流程就会一步步「演」给你看，而不是一次性堆给你。

核心特色

FEATURES · 六大亮点

逐步可视化

每个知识点拆成多个步骤，点击「下一步」即可看到概念「动起来」，信息不一次性倾倒。

时间轴叙事

用纵向时间轴讲述 NLP 从 1940s 到深度学习时代的发展脉络，徐徐展开。

架构解剖

Transformer 各组件分层透视，数据流向一目了然，把大模型拆成零件逐个点亮。

沉浸式暗色主题

护眼的深色界面搭配标志性薄荷绿高亮，长时间阅读不疲劳。

零构建零依赖

无 Node、无 Webpack、无 npm install——纯静态文件，双击 HTML 即可运行。

响应式适配

桌面 / 平板 / 手机三端自适应，移动端可折叠侧栏，随时随地学习。
